== Installation ==

1. Please upload the entire `H Seeed WP` folder to the /wp-content/plugins/ directory.
2. Plase activate the plugin through the Plugins menu in WordPress.

1.管理画面のプラグイン追加ページからH Seeed WPと検索しダウンロード、有効化してください

<h3>Else</h3>

1. Plase search and install H Seeed WP in WordPress Plugins menu.
2. please activation it.
